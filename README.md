# Java Swing To-Do App

A simple desktop To-Do List application built using **Java Swing**.

## Features
- Add tasks
- Delete tasks
- Scrollable task list

## Technologies
- Java
- Swing (GUI)

## How to Run
1. Clone the repo
2. Compile: `javac ToDoApp.java`
3. Run: `java ToDoApp`
